# Sony TV YouTube Timeout Defeater

One Paragraph of project description goes here

## Installing

```
npm install -g
```

## CLI Example

```
youtube-timeout-defeater --host=192.168.19.61
```

### Options
* host - TV IP address (internal)
* port - Defaults to 80
* authkey - TV pre-shared key (defaults to '0000')
